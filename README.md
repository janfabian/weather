Weather forecast
=================================
1. Copy api.js.example into api.js

    ```cp js/models/api.js.example js/models/api.js```
2. Fill [google key](https://code.google.com/apis/console) and [weather key](http://www.worldweatheronline.com/) in ```js/models/api.js```
3. Install libraries

    ```bower install```
4. Compile scss

    ```compass compile```
5. Start web server 

    ```http-server ```
